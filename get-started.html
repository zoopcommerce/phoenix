Getting Started

Custom templates for your Zoop online store

Download

Before downloading, be sure to have a code editor (we recommend Sublime Text 2) and some working knowledge of HTML and CSS. We won't walk through the source files here, but they are available for download. We'll focus on getting started with the compiled Bootstrap files.

File structure

The default file structure required by the Zoop templating engine is as follows: 

    phoenix/
    ├── assets/
    │   ├── css/
    │   ├── images/
    │   ├── js/
    │   ├── twig/
    ├── layouts/
    ├── macros/
    ├── cart.twig
    ├── checkout.twig
    ├── collection.twig
    ├── contact.twig
    ├── download.twig
    ├── index.twig
    ├── maintenance.twig
    ├── product.twig
    ├── search.twig
    └── thankyou.twig

The included *.twig files are required as they correspond directly to a page or section in your store. You can add as many other assets as you require within the assets, layouts and macros folders.
